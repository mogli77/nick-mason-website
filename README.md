# Nick Mason Construction, Inc. — Website

A custom website for Nick Mason Construction, built with clean HTML, CSS, and JavaScript.

## Project Structure

```
nick_mason_website/
├── index.html          ← Main homepage
├── css/
│   └── styles.css      ← All styling
├── js/
│   └── main.js         ← Interactive functionality
├── images/             ← Your photos go here
├── videos/             ← Your video loops go here
├── .gitignore          ← Files Git should ignore
└── README.md           ← This file
```

## Quick Start

1. Open `index.html` in your browser to preview the site
2. Edit `index.html` to change content (text, structure)
3. Edit `css/styles.css` to change appearance (colors, fonts, spacing)
4. Add your photos to the `images/` folder
5. Add your video loops to the `videos/` folder

## Making Changes

**To change colors:** Edit the CSS variables at the top of `styles.css`
**To change fonts:** Update the Google Fonts link in `index.html` and the font variables in `styles.css`
**To add projects:** Copy a `<article class="project-item">` block and update the content

## Deployment

This site is deployed via Netlify, connected to this GitHub repository.
Any changes pushed to the `main` branch will automatically deploy.
